# Minikick

Mini Kickstarter, per the specification [here](https://gist.github.com/ktheory/3c28ba04f4064fd9734f).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'minikick'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minikick
